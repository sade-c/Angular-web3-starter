Angular-web3-starter

Basic operations for those who want to develop web3 on Angular.

Template

Free Angular Admin Template by PrimeNG www.primefaces.org/sakai-ng

Usage

The following is a basic example of how to use the Angular-web3-starter.

The src/app/app.module.ts file contains the main Angular module and the src/app/app.component.ts file contains the main component.

The src/app/app.component.html file contains the HTML template for the main component.

The src/app/app.component.css file contains the CSS styles for the main component.

The src/app/app.component.ts file contains the Angular component class.

The src/app/app.component.html file contains the Angular component HTML template.

Installation

The following is a basic example of how to install the Angular-web3-starter.

The npm install command is used to install the Angular-web3-starter.

The npm run build command is used to build the Angular-web3-starter.

The npm run serve command is used to serve the Angular-web3-starter.

The npm run watch command is used to watch the Angular-web3-starter.

The npm run test command is used to test the Angular-web3-starter.

The npm run build.polygon command is used to polygon environment Usage

The npm run build.avalanche command is used to avalanche environment Usage
